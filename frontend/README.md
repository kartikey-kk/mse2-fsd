# Student Authentication Frontend

React frontend for the MERN Student Authentication System.

## Features

- Register student
- Login with JWT
- Protected dashboard
- Update course
- Update password
- Secure logout

## Environment

Create `.env` in this folder and set:

`VITE_API_BASE_URL=https://your-render-backend.onrender.com/api`

## Run

1. `npm install`
2. `npm run dev`

## Build

`npm run build`
